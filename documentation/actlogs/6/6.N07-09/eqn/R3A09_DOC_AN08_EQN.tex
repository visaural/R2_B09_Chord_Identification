		\textbf{Chord number}
	\end{center}
	
	The chord number $N_C$ for any given chord with root note number $R$ and chord type number $S$ is given by:
	\begin{equation*}
		N_C = 37R + S \,,
	\end{equation*}
	
	where $N_C$, $R$, and $S$ are integers, and $R \in [0, 11]$, $S \in [0, 36]$, and $N_C \in [0, 443]$.\\
